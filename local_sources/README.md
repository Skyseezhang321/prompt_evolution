# Local Source Snapshots

This directory is for local-only source snapshots used for research traceability.

Rules:

- Put raw third-party text, screenshots, PDFs, and HTML exports under `local_sources/raw/`.
- `local_sources/raw/` is ignored by git and should not be committed.
- Tracked notes should record only the source URL, capture date, local snapshot path, SHA256 hash, and structured summaries.
- For copyrighted or login-required sources, keep quotes short and write conclusions as paraphrases.
