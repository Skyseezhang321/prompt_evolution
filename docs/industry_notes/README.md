# 行业经验笔记

本目录记录值得深读的行业经验来源，例如高质量工程博客、知乎文章、Twitter/X thread、产品文档、会议分享、issue 讨论和事故复盘。

使用约束：

- 外部贡献先按 [共创工作流](../contribution_workflow.md) 提交或认领，低门槛线索不要求直接写完整笔记。
- 所有线索先登记到 [来源清单](../source_inventory.md)，再决定是否进入本目录深读。
- 每个重要来源使用 [template.md](template.md) 建一份独立笔记，文件名优先使用 `source_inventory.md` 中的 `source_id`。
- 本目录记录单一来源的结构化观察；跨来源综合判断写入 [行业实践整理](../industry_practices.md)。
- 明确区分作者主张、可观察证据、本项目推测和已验证结论。
- 对知乎、Twitter/X、付费文章等来源，只保留必要短摘录、链接、位置和摘要；不要整篇转载受版权保护内容。
- 用户粘贴、登录后可见或无法稳定公开访问的原文，优先保存到 `local_sources/raw/` 作为本地私有快照，并在笔记中记录路径和 SHA256；不要把原文全文提交到 git。
